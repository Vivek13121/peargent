# Installation

## Prerequisites

- Python 3.8 or higher
- pip package manager

## Install via pip

```bash
pip install peargent
```

## Install from source

```bash
git clone https://github.com/yourusername/peargent.git
cd peargent
pip install -e .
```

## Optional Dependencies

### For PostgreSQL support

```bash
pip install psycopg2-binary
```

### For Redis history support

```bash
pip install redis
```

### For SQLAlchemy (database tracing)

SQLAlchemy is included by default in the latest version.

## Verify Installation

```python
import peargent
print(peargent.__version__)
```

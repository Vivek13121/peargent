# Welcome to Peargent Documentation

Peargent is a modern Python framework for building intelligent AI agents with advanced tracing, history management, and seamless LLM integration.

## Quick Start

Install Peargent using pip:

```bash
pip install peargent
```

## Create Your First Agent

```python
from peargent import create_agent
from peargent.models import groq

agent = create_agent(
    name="assistant",
    persona="You are a helpful AI assistant.",
    model=groq("llama-3.3-70b-versatile")
)

result = agent.run("Hello, world!")
print(result)
```

## Key Features

- **Advanced Tracing**: SQLAlchemy-based tracing with async writes and custom table names
- **Smart History**: Flexible conversation history with PostgreSQL, SQLite, Redis support
- **Multi-LLM Support**: OpenAI, Groq, Gemini, and more with automatic cost tracking
- **Async Database**: Non-blocking database writes for better performance
- **Custom Tools**: Easy tool creation and management

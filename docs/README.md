# Peargent Documentation

This is the documentation site for Peargent, built with Next.js and Nextra.

## Development

```bash
npm install
# or
pnpm install

npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

The documentation is available in the `/pages` directory and uses Nextra for rendering.

## Structure

- `/app/page.tsx` - Landing page
- `/pages` - Nextra documentation pages
- `/theme.config.tsx` - Nextra theme configuration
- `/public` - Static assets

## Build

```bash
npm run build
npm run start
```

## Deploy

Deploy to Vercel, Netlify, or any Next.js hosting platform.
